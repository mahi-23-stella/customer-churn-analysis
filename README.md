# customer-churn-analysis
A Power BI-style interactive dashboard for analyzing customer churn in a telecom dataset.

Customer churn is one of the biggest challenges in the telecom industry. This project analyzes 100 customer records to identify key churn patterns, high-risk segments, and retention opportunities.
MetricValue
📦Total Customers        100 
🔴 Churn Rate            52%
🟢 Retention Rate        48%
💰 Avg Monthly Charge    ₹82
📅 Avg Tenure            34 months

 Key Findings
🔴 Month-to-month contract customers churn 89% more than others
⚡ Electronic check users have the highest churn risk (78%)
📞 Customers with 5+ support calls have 75%+ churn probability
⏱️ Low tenure (< 12 months) = highest churn segment (82%)
🌐 Fiber Optic internet users churn more (68%) than DSL users
👶 Age group 20–30 has the highest churn rate among all age segments
🛡️ Two-year contract + Tech Support customers are the most loyal
