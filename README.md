# 📊 Customer Churn Analysis Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard to analyze customer churn in a telecom dataset. The objective is to identify key churn patterns, high-risk customer segments, and opportunities to improve customer retention using data-driven insights.

---

## 📊 Key Metrics

| Metric                | Value     |
| --------------------- | --------- |
| 📦 Total Customers    | 100       |
| 🔴 Churn Rate         | 52%       |
| 🟢 Retention Rate     | 48%       |
| 💰 Avg Monthly Charge | ₹82       |
| 📅 Avg Tenure         | 34 months |

---

## 🔍 Key Insights

* 🔴 Customers with **month-to-month contracts** show significantly higher churn (~89%) compared to long-term contracts
* ⚡ Users paying via **electronic check** have the highest churn risk (~78%)
* 📞 Customers with **5+ support calls** show a churn probability above 75%
* ⏱️ Customers with **low tenure (< 12 months)** are the most likely to churn (~82%)
* 🌐 **Fiber Optic users** have higher churn (~68%) compared to DSL users
* 👶 The **20–30 age group** shows the highest churn among all segments
* 🛡️ Customers with **two-year contracts and tech support** are the most loyal

---

## 🛠️ Tools & Technologies

* Power BI
* CSV Dataset
* Data Cleaning & Visualization

---

## 🚀 Conclusion

The analysis highlights key factors driving customer churn, such as contract type, tenure, payment method, and service usage. These insights can help businesses design targeted retention strategies and improve customer satisfaction.
